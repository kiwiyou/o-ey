A user translation viewer for BOJ

For the actual translation content, see [kiwiyou/boj-user-translation.](https://github.com/kiwiyou/boj-user-translation)

## Install

```
$ npm install
```

## Development

```
npm run dev chrome
npm run dev firefox
npm run dev opera
npm run dev edge
```

## Build

```
npm run build chrome
npm run build firefox
npm run build opera
npm run build edge
```
